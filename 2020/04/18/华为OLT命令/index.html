命令模式关系图

命令模式特点
  • 向下兼容
    普通用户模式中的所有命令在特权模式下都能执行；普通用户模式和特权模式中的所有命令在全局配置模式下都能执行。还有一些命令在任何模式下都能执行，如display board。
  • 分级保护
    可以防止未授权的用户的非法侵入。不同级别的用户可以进入不同的命令模式。同时，对于不同级别的用户，即使进入同样的模式，他们所能执行的命令也会有所不同。

机框号/槽位号/端口号

机框号（frameid）

使用的机框号以主框为例，从框不单独介绍。

机框类型 主控板 机框号取值
MA5680T/MA5683T SCUN/SCUK 0-32，其中 0 表示主框，1-32 表示 MA5623AR 从框。
槽位号（slotid）

使用的槽位号以大规格设备 MA5680T ETSI 业务框为例，其他业务框不单独介绍。

业务框类型 槽位数 通用接口板槽位 主控板槽位 业务板槽位 上行接口板槽位 电源板槽位
MA5680T ETSI 业务框 23 0 9， 10 1-8， 11-18 19， 20 21， 22
MA5680T IEC 业务框 21 0 7， 8 1-6， 9-16 17， 18 19， 20
MA5683T 13 12 6， 7 0-5 8，9 10， 11
端口号（portid）

自己看板卡有几个口就有几个口：display board 0/11（frameid / slotid）

display ont info by-loid 1234567890 { by-desc description | by-ip ip-address | by-loid loid-value | by-mac mac-value | by-password { password | hex password } | by-sn sn-value | frameid { slotid { portid { ontid | all } | all } }

例： display ont info 0 2 4 10 查询 frameid框 0 slotid槽位 2 portid端口 4 第 10 个ONT 。
